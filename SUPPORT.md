# Support

## 📚 Documentation

Check the [docs](docs/) directory for guides and references.

## 🐛 Bug Reports

Open an issue on [GitHub](https://github.com/mohameden19961/codeguard/issues) with:
- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- CodeGuard version (`codeguard --version`)

## 💡 Feature Requests

Open an issue with the `enhancement` label including:
- Clear description of the feature
- Use case or motivation
- Potential implementation approach (optional)
